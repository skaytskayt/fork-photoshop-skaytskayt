# GrayBit Image Editor

Единое приложение — редактор изображений (ТКГ, лабораторные 1–5).

**Демо на хостинге:** photoshop-lopaev-kiril

## Функциональность

- **Загрузка и сохранение** (лаб. 1): PNG, JPG и собственный формат GB7; информация о файле в статус-баре
- **Цветовые каналы и пипетка** (лаб. 2): панель каналов с миниатюрами и включением/выключением, пипетка с RGB и координатами пикселя
- **Гистограмма и «Уровни»** (лаб. 3): коррекция тонового диапазона по каналам с гистограммой и предпросмотром
- **Масштабирование** (лаб. 4): изменение размера изображения (ближайший сосед / билинейная интерполяция), масштаб отображения в статус-баре
- **Фильтрация ядром свёртки** (лаб. 5): сетка 3×3, предустановки (резкость, Гаусс, размытие, оператор Прюитта), стратегии краёв, вычисление в Web Worker

## Запуск

```bash
npm install
npm run dev      # режим разработки
npm run build    # production-сборка в dist/
```

Деплой на GitHub Pages выполняется автоматически при пуше в `main`.
