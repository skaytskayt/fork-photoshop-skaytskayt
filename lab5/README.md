# Лаб. 5 — Фильтрация изображений ядром свёртки

Расширение GrayBit Image Editor: фильтрация через ядра свёртки 3×3.

## Функциональность

- Модальный диалог «Фильтр…» с сеткой 3×3 для ввода коэффициентов ядра
- Предустановленные ядра: тождественное, повышение резкости, Гаусс 3×3, прямоугольное размытие, оператор Прюитта (X и Y)
- Выбор каналов: R, G, B (и A для изображений с прозрачностью)
- Стратегии заполнения краёв: чёрный, белый, копирование
- Предпросмотр в реальном времени
- Вычисление в Web Worker — интерфейс остаётся отзывчивым при обработке больших изображений
- Сохранение в PNG, JPG и GB7
